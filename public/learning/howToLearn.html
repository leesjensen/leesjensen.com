How to learn

Everyone learns differently. However, there are some common patterns that are useful to observe.

  1. Environment - Patrick Lencioni
    1. Trust: Everyone trusts each other
    2. Critical discussion: Everyone openly shares their ideas and thoughts
    3. Commitment: Everyone is committed to each others success
    4. Accountability: Everyone holds each other accountable
    5. Goal: Everyone is working towards the same goal
  2. What to learn - Roger Schank
    1. How to be a critic
    2. Whom to respect and copy
    3. How to know where you fit
    4. How to take action
    5. How to think
  3. Find joy in the journey
  4. Let curiosity take you
  5. Embrace failure
  6. Anything that is good takes work
